# StockMyGhar 📦🛒

**StockMyGhar** is an offline-first, locally-hosted grocery and household inventory application. Designed with a mobile-first philosophy, you can access your home inventory through an intuitive responsive visual dashboard right from your smartphone, as long as it's connected to your local network or hotspot!

---

## ✨ Supercool Features

* **📱 Mobile-First Experience**: A premium, responsive design that works beautifully on any device.
* **🗃️ Container Optimization**: Group your groceries logic using "Containers" (e.g., Refrigerator, Pantry, Cleaning Closet) customized with colors and icons!
* **📝 Detailed Item Tracking**: Specify exact quantities, units (e.g., pcs, kg, boxes), specific notes, and upload images straight from your phone camera. 
* **📸 Local Image Hosting**: Never lose track of an item. Photos are elegantly processed via Base64 logic and efficiently saved onto the server's local disk.
* **🕵️ Database Audit Logging**: Track *all* operations! A robust built-in `AuditLog` captures exactly who/when/what added, updated, or deleted a container and its items. 
* **⚡ Blazing Fast SQLite**: Zero cloud overhead. Database is saved directly to `stockmyghar.db`.

---

## 🛠️ Technology Stack

* **Core Engine**: Python 3 & Flask
* **Database & ORM**: SQLite + SQLAlchemy (`Flask-SQLAlchemy`)
* **Styling & UI**: Vanilla CSS & JavaScript + HTML Jinja2 Templates
* **Architecture**: Offline-first Host implementation

---

## 🚀 How to Run It!

1. **Install Dependencies**:
   Ensure you have Python installed, then run:
   ```bash
   pip install -r requirements.txt
   ```

2. **Launch the Server**:
   Simply execute the specialized Windows batch file to trigger the server setup on your local IP.
   ```bash
   run_server.bat
   ```

3. **Access Your Pantry**:
   * On your PC: Navigate to `http://127.0.0.1:5000`
   * On your Phone: Connect your phone to the SAME Wi-Fi or PC Mobile Hotspot, and navigate to the local IP outputted in the terminal (e.g., `http://11.55.222.444:5000` or whatever the console tells you!).

---

## 📂 Project Architecture

* `app.py`: The powerhouse behind the server APIs, ORM Database models (`Container`, `Item`, `AuditLog`), and template rendering.
* `stockmyghar.db`: The standalone SQLite database automatically generated by SQLAlchemy.
* `run_server.bat`: The ultimate zero-friction launch execution protocol.
* `static/uploads/`: The robust disk-storage solution for item pictures.
